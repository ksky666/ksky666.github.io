- CSS
- JavaScript

## How to Navigate
- `index.html` is the homepage (about, education, skills, projects, contact).
- `visualizations.html` contains two SVG outputs generated by JavaScript:
  1) an SVG visualization (bar chart)
  2) a creative SVG artwork (random stars)
